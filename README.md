# CFD Design and Simulation of a 600 N Rocket Nozzle

## Overview
This project presents the design and CFD simulation of a convergent–divergent rocket nozzle capable of producing approximately 600 N thrust using ANSYS Fluent.

The nozzle was designed using compressible flow equations and validated using CFD analysis.

---

## Design Conditions

Chamber pressure: 1.8 MPa  
Chamber temperature: 3000 K  
Exit pressure: 0 Pa  
Gas model: Ideal gas  
Turbulence model: k–ω SST  

---

## Nozzle Geometry

Chamber diameter: 48 mm  
Throat diameter: 16 mm  
Exit diameter: 36.6 mm  
Total length: 226 mm  
Area ratio: 5.23  

---

## CFD Results

• Choked flow achieved at throat  
• Exit Mach number ≈ 2.8  
• Pressure drops from 1.2 MPa to 0.1 MPa  
• Temperature drops from 3300 K to 1700 K
